# DXMarkers

#### 介绍
DXMarkers 是一款针对单细胞手动注释细胞类型的 R 包，主要用于Marker的识别和检索，帮助确定细胞类型。用户可以在"CellMarker"、"CellMarker2.0" 和 "PanglaoDB"三大数据库中，高效查找Marker及其对应细胞类型。DXMarkers 提供了基于物种和组织类型的筛选功能，使Marker检索更为精确，为医学生、临床医生、科研工作者节省时间，提高效率。
